Filenames are set in config/portal.php. Current files:

    secretary.png           Sh. Rajesh Goel               300x346

Source
------
Downloaded from the official HSBTE website (https://hsbte.org.in/) on
22 July 2026. These are the board's own official photographs, shown here at the
same 150px circular size the official site uses.

Note: the Secretary's file is served as "scey.jpg" on the official site but is
actually PNG data, so it is stored here with the correct .png extension.

Known quality limitation
------------------------
chairman.jpg is only 132x141 and is displayed at 150px, so it is slightly
upscaled and will look soft on high-resolution screens. The official site has
the same limitation. Replace it if a larger version becomes available.

Replacing a photo
-----------------
- Square images work best. They are cropped to a circle, aligned to the top so
  faces are not cut off.
- 400x400 px or larger, ideally under ~200 KB.
- If the file extension changes, update the matching entry in config/portal.php.

If a file is missing the portal falls back to an initials avatar, so the layout
will not break while photos are being swapped.

Keep this list current -- ministers and IAS officers get transferred, and a
wrong name or photo on a government portal is very visible.
